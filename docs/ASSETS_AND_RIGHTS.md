# 素材・権利の確認（販売前）

ZIP や有料配布をする前に、**自分以外の素材**が混ざっていないか・**ライセンスで販売できるか**を確認してください。

## このプロジェクトで確認しやすい場所

| 種類 | 場所 | メモ |
|------|------|------|
| BGM | `public/bgm/*.mp3` | **作者メモ**: Snow.ai の**課金プラン**で生成した音源を使用している場合、当該サービス**当時の利用規約**に従いゲーム内再生・ZIP 同梱での再配布が許される範囲で利用する。契約・請求の記録（スクショ等）を保管しておくとトラブル時に安心。規約改定時は公式の最新版を確認すること。 |
| アイコン | `public/icon.svg` 等 | 同様 |
| フォント | `TitleScreen` の `next/font`（Google Fonts） | 通常は OFL だが、**商品に同梱するフォントファイルがある場合**は別途確認 |

## 最低限やること

1. **各ファイルの出所**（自作 / 素材サイト / 友達制作）をメモする  
2. **再配布・商用・クレジット表記**が許されているかサイトの規約で確認  
3. 必要なら **クレジット** を商品説明 or 同梱 `CREDITS.txt` に書く  

## リポジトリの LICENSE

`LICENSE` が無い場合でも、**販売用 ZIP に含めるファイル**については、上記の素材ごとの権利が別に効きます。迷ったら素材元の規約に従うのが安全です。
