# 特定商取引法に基づく表記（BOOTH 用テンプレート）

**このファイルは法的助言ではありません。** 空欄 `【】` を自分の情報で埋め、BOOTH の「特定商取引法に基づく表記」欄に貼り付けてください。内容は **事業の形態（個人／法人）** で要領が変わるため、迷ったら税理士・法務に確認してください。

---

## 貼り付け用（そのまま編集して使う）

```
■販売業者名
【氏名または屋号。法人なら法人名】

■運営統括責任者
【個人販売なら自分の氏名。法人なら代表者名など BOOTH の案内に従う】

■所在地
【請求があれば遅滞なく開示できる住所。個人で住所を出したくない場合は BOOTH の案内と個人情報保護の両方を確認のうえ対応】

■電話番号
【連絡用。個人で番号を出したくない場合は BOOTH の運営案内を参照】

■メールアドレス
【問い合わせ用の受け取れるアドレス】

■商品代金以外の必要料金
消費税（表示価格に含む場合は「含む」と記載）
インターネット接続料金、通信料金等はお客様の負担となります。

■申込の有効期限
特になし（在庫のあるダウンロード商品のため）。

■不良品の取扱い
デジタルコンテンツの性質上、ファイル破損・誤配布など明らかな提供不備がある場合に限り、合理的な範囲で再ダウンロード等に対応します。詳細はお問い合わせください。

■販売価格
各商品ページに税込みで表示しています。

■支払方法
BOOTH が提供する決済方法に準じます（クレジットカード等）。

■支払時期
注文時に決済が行われます。

■引渡時期
決済完了後、BOOTH の画面からダウンロード可能になります（即時〜数分程度が目安）。

■返品・キャンセル
デジタルデータの性質上、ダウンロード開始後のお客様都合による返品・返金はお受けできません。BOOTH の規約・キャンペーン条件がある場合はそちらに優先することがあります。
```
